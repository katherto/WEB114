The Differences Between =, ==, and ===

"=" is an assignment operator in JavaScript. It's used to assign objects, functions, and values to variables.
"==" is a comparison operator in JavaScript. It's used to compare two values and will return a true or false.
"===" is also a comparison operator in Javascript. While comparing for equality like "==" it takes it a step further and compares object types then evaluates as true or false.
